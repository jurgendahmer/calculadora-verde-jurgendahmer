# 🌱 Calculadora Verde

Página interativa que incentiva hábitos sustentáveis através de cálculos automáticos e mensagens educativas. Projeto desenvolvido com **HTML**, **CSS** e **JavaScript puro** (sem bibliotecas externas).

## ✨ Funcionalidades

- Cálculo de litros de água economizados por mês
- Cálculo de CO₂ evitado com reciclagem e lâmpadas LED
- Cálculo de árvores necessárias para compensar o CO₂ evitado
- Pontuação ecológica (0–100) com barra de progresso animada
- Ranking ecológico: Iniciante Verde, Amigo da Natureza, Guardião do Planeta
- Mensagens motivacionais aleatórias
- Tema claro/escuro
- Som de confirmação ao calcular
- Histórico de resultados salvo no navegador (localStorage)
- Layout responsivo (funciona em celular)

## 🚀 Como usar

Basta abrir o arquivo `index.html` em qualquer navegador. Não é necessário instalar nada.

## 🧮 Como os cálculos funcionam

- **Água:** cada minuto reduzido no banho economiza ~12 litros, multiplicado por 30 dias.
- **Reciclagem:** cada kg reciclado evita ~5 kg de CO₂.
- **Energia:** cada lâmpada LED evita ~0,5 kg de CO₂ por mês.
- **Árvores:** cada árvore absorve ~22 kg de CO₂ por ano, usado para calcular quantas árvores compensariam o CO₂ evitado em um ano.
